# 💼 Loan Prediction Dashboard

An intelligent machine learning dashboard built with Streamlit that predicts loan approval decisions.

## 🚀 Features

* 📊 Data visualization
* 🔍 Smart filtering system
* 🤖 Machine learning model (Random Forest)
* 📈 Model evaluation (Accuracy, F1, ROC)
* 🧠 Real-time prediction simulator

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Plotly

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📂 Dataset

Upload your own dataset (CSV) using the sidebar.
